System Architecture 3

The system integrates collaborative filtering and Machine Learning models, processing federal contract data to recommend relevant partners.

  • Data ingestion: Via API from SAM.gov to Microsoft Fabric platform
  • ETL Pipeline: Data is loaded into a Lakehouse as CSV, cleaned, and stored in delta tables
  • Filtering: Records filtered by award amount and timeframe for optimal targeting
  • Synapse dataflow: Prepares data for Power BI reporting


Pipeline Workflow

  1. SAM.gov API to Lakehouse: Copy contract data to staging tables
  2. Auto-refresh Notebook: Adds timestamps for updates
  3. 1-Month Filter Notebook: Filters data by recent awards
  4. 3-Year Filter Notebook: Filters awards with = $2 million value over three years
  5. Merged Notebook: Combines filtered records
  6. Basecamp Automation: Updates project docs with CSV outputs
  7. Failure Notification: Sends email alerts for pipeline failures
